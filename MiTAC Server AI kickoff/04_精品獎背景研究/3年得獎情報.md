## 🔍 從群聯金獎介紹歸納 5 條 wording 黃金法則

### 法則 1:**開場句 = 動詞 "引領" + 技術突破 "首個"**
- ✅ 群聯:「群聯**正在引領**人工智慧產業,提供了**首個**解決 X 問題的方案」
- ❌ 不要:「我們**提供**先進的 X 解決方案」← 平淡

### 法則 2:**技術名稱 OK,客戶名稱不要**
- ✅ 提:Llama 7b / 700 億參數 / NAND / aiDAPTIV+ / aiDAPTIVlink(技術 / 自家產品名)
- ❌ 不提:「服務 OpenAI / Anthropic / Meta」(客戶名不出現)

### 法則 3:**用「業界痛點」做 framing**
- ✅ 群聯:「中小型企業僅能進行小規模且不確實的訓練,**並無法擴展到 Llama 7b 以上**」
- → 把市場痛點具體量化(7b 是門檻)

### 法則 4:**「軟硬體混合 / 混合式」是 2025 金獎熱字**
- ✅ 群聯用 2 次:「**混合式**解決方案」「**軟硬體混合型**解決方案」
- → 對應 Server AI:邊緣 + 雲端 = **天然軟硬體混合**!直接抄這個 framing

### 法則 5:**「地端 / 資料中心等級」對齊算力下沉趨勢**
- ✅ 群聯:「讓您**在地端**獲得過去**僅限於資料中心等級**的推論及運算效能」
- → 對應 Server AI:**讓邊緣行車記錄器獲得過去僅限於資料中心等級的 AI 視覺分析**

> **研發 / 設計 / 品質 / 行銷 / 台灣產製(MIT)**

## 📚 額外 AI 金獎深挖案例(d 任務產出)

### Case 1:**雲象科技 aetherAI Hema** — 醫療影像 AI(跟 Server AI 最像)
- 產品:全球首個 AI 輔助骨髓抹片分類系統
- 關鍵 wording:
  - 「**世界首個**(world's first)同時取得衛福部 + 歐盟 CE 雙認證」
  - 「訓練於**全球最大資料集(超過 100 萬個細胞)**」
  - 「**平均準確率 94%**」
- 拿過獎:2022 台北生技獎金獎 + 經濟部新創獎金獎
- 🎯 **學到 4 條 wording**:
  1. **「全球首個 X 同時取得 A + B 雙認證」**(雙認證敘事 = 評審大愛)
  2. **「訓練於全球最大資料集 N 筆」**(數據規模佐證)
  3. **「平均準確率 X%」**(量化指標)
  4. **「Asia's leading medical image AI solution provider」**(自封 leader 但要技術背書)

### Case 2:**雃博 智慧 IOT 壓傷照護氣墊床**(2025 金獎)
- 產品:Wellell 智慧 IOT 全時壓傷照護減壓氣墊床
- 公司 framing:「**世界級醫療品牌,銷售通路與服務網路遍及 70 餘國**」
- 品牌敘事:「秉持『尊重生命價值』的信念」
- 🎯 **學到 3 條**:
  1. **「世界級 X 品牌」** + **「遍及 N 國」**(規模佐證 — 我們也可以說「跨亞美歐澳 4 區」)
  2. **品牌信念句**(類似 ESG / 永續 framing)
  3. **公司歷史**(「自 1990 年成立以來」— MiTAC 1989 比這還久,可以用)

### Case 3:**SysTalk.Chat 智能客服機器人**(2023 銀獎)
- 產品:雙腦一流程交談式 AI 客服
- 關鍵 wording:
  - 「**獨創**雙腦一流程」(獨特技術名)
  - 「**高達 90% 以上**的語意分析精準度」
  - 「**獲國內多家銀行、政府機關、企業採用導入**」(籠統客戶但講出領域)
- 🎯 **學到 2 條**:
  1. **「獨創 + 自家技術名」**(把架構命名為自家專屬詞 — Server AI = aiDAPTIV+ / aetherAI Hema / SysTalk 同 pattern)
  2. **「N% 精準度」量化 + 客戶領域不點名**

### Case 4:**digiLogs 集中日誌管理平台**(SaaS 銀獎)
- 「同時管理 **55 種以上**不同的 Log 格式」
- 🎯 **學到 1 條**:**用「N 種以上」做數字佐證**(整合能力具體化)
