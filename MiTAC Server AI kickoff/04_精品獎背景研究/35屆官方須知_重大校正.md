## 🌱 「永續產品特色」200 字選填補充欄位 — 6 項條件(全要符合 + 非漂綠)

如果要填這 200 字選填區,必須符合以下 6 項條件:

1. **產品 + 包裝採用綠色環保 / 安全無害 / 循環再生材料**
2. **生命週期 + 廢棄物處理**;耐用、可靠、可重複利用、可升級、可修復
3. **節能 / 提升能源效率**
4. **已申請或取得碳足跡認證、碳盤查相關認證** ← MiTAC 有嗎?
5. **使用後帶來環境 / 社會 / 使用者行為的永續改善**
6. **製程中保障勞工 / 性別平等 / 社會責任**

→ **非漂綠** = 所有「永續、綠色、減碳、碳中和」聲明都要有國際或本國認證體系背書、可供查詢核實
→ **參考依據:歐盟 ESPR (Ecodesign for Sustainable Products Regulation)**

## 🎯 評選項目的 24 個子項、67 個細項

5 大評選項目細部展開(每個都要逐一回應,不能空白):

### 研發(5 個子項)
1. 研發策略朝向提升該產品之獨特性與市場競爭力布局(4 細項)
2. 核心研發能量布局完善(3 細項)
3. 智慧財產管理制度及成效(2 細項)
4. 該產品在使用操作上考慮到安全性與便利性(2 細項)
5. 該產品在製程上節省成本與創新,並考慮環保及綠能製程(3 細項)

### 設計(5 個子項,各細項較簡短)
1. 具備適切的功能,符合市場需求
2. 使用方便、舒適、安全 + 人因工程 + **包容性設計與中性語言,避免性別刻板印象**
3. 精湛的工程技術,產品性能優異
4. 精緻優美之外形
5. 考量環保設計,符合環保規範

### 品質(5 個子項)
1. 魅力品質、創新品質(4 細項)
2. 嚴格控制設計、製造過程品質(4 細項)
3. 獲得 ISO 9001 等驗證且確實執行(4 細項)
4. 落實 TQM(4 細項)
5. 品質經營層次(**ESG 在這裡!**)(5 細項)

### 行銷(5 個子項)
1. 推廣產品核心價值及品牌特性 + 產品定位(7 細項)
2. 國內外銷售通路管理(6 細項)
3. 競品資訊收集 + 顧客滿意度調查(6 細項)
4. 顧客服務系統(訴怨處理 + 維修)(5 細項)
5. 品牌推廣具體作法 + 預算 + 專人(5 細項)

### 臺灣產製(4 個子項,**400 字**)
1. 參選產品原產地是否為我國製造
2. 是否有原產地證明書 / 出口報單 / MIT 微笑標章
3. 部分在我國取得或製造的比例
4. 完全在國外製造則說明製造國

## ⚠️ 評審常見 NG vs DO

### ❌ NG(我們要避免)
- **缺乏統籌** — 行銷新人寫技術內容失準
- **佐證不足** — 認驗證、專利、得獎證明未整理引用
- **未對應評審項目** — 自己愛寫什麼寫什麼,沒按子項細項回應
- **浮誇無實證** — 形容詞堆砌、缺數據實例競品分析
- **行銷說明空泛** — 無具體作法、數據或成效
- **定位不清** — 品牌與產品無明確市場調查
- **內容與產品不符** — 文件誇大,實品評鑑時失分 ← **要送實品**

### ✅ DO(我們要做的)
- **組建團隊** — 各部門協助撰寫,**高層統籌指揮**
- **顧客視角** — 把審查委員當「第一次接觸的重要顧客」
- **完整回應** — 67 細項都要逐一回應,確保資訊清楚有條理
- **論述具體簡明準確** — 避免冗長空泛
- **客觀數據優先** — 市場調查、競品分析、專利、實測數據
- **突顯創新獨特性** — 與現有解決方案差異
- **對應產業趨勢** — ESG、AI、IoT、智慧製造 ← Server AI 完美命中
- **行銷策略具體** — 推廣方式、應用場景、具體數據驗證
