Typography

8 階 type scale。不要再加 13.5px / 14.5px / 16.5px 半 px 值;從這 8 個之中挑。

3xl · 40px Cabinet Grotesk 800 Hero / H1
2xl · 30px Cabinet Grotesk 800 H2 / Page title
xl · 24px Cabinet Grotesk 700 H3 / Zone title
lg · 20px Cabinet Grotesk 700 H4 / Section heading

base · 15px Satoshi 400 Body
sm · 13px Satoshi 500 Small / Button label
xs · 11px Satoshi 600 Caption / File label / Meta

Font families

  • --font-display = Cabinet Grotesk · 標題、Hero、navigation。Fontshare CDN
  • --font-body = Satoshi · 內文、卡片、按鈕。Fontshare CDN
  • --font-mono = SF Mono / Consolas · 程式碼、檔名、Jira ID

Cards · BEM Naming(legacy class kept as alias)

主元素 .card + modifier .card--featured / .card--compact / .card--link / .card--status-*。

Featured card

.card .card--featured · 主要內容區大卡。Hero zone / 本週 summary。

Compact

.card .card--compact · 小卡 / stat 卡片。

Link card

Accessibility Rules

  • 所有 emoji 必須包 <span aria-hidden="true">(已對 791 個 emoji 自動加完)
  • 所有互動元素 必須有 :focus-visible 視覺(全域已加在 design-tokens.css)
  • Color 對比:文字 + 背景必須 ≥ 4.5:1(AA);大字 ≥ 3:1。--text-mute-2 是允許用的最淺值
  • Icon-only links 必須有 aria-label
  • 動畫:prefers-reduced-motion 已全域 honor(在 design-tokens.css)
  • Print:已配 print stylesheet,列印自動 hide nav / chrome,只剩內容
