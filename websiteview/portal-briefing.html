Presentation 中,這 5 題會在前 5 分鐘冒出來。先把答案準備好。

Copyright © MiTAC Digital Technology Corp.
Q1 · Q2 — 哪個 portal,看到什麼
PART 2 · FAQ●
Q1 · 我用哪個 portal?(依角色)
經銷商 / 神達內部Master Portal — 看到所有客戶的設備池
車隊老闆 / 司機主管Fleet Portal — 只看到自己車隊
同時管多個車隊的 IT兩個都要 — Master 派發、Fleet 運營
Q2 · 我能看到什麼?
  • 行程(Trip):每台車每天的開車紀錄(時間、里程、軌跡、駕駛是誰)
  • 事件(Event):5 種類型 — 急煞、跟車過近、超速、ADAS 校正、DMS 校正
  • 影片:事件當下前向 + 駕駛艙雙鏡頭片段
  • 報表:駕駛安全評分、車隊每日活動量、設備健康度
Copyright © MiTAC Digital Technology Corp. 6 / 25
Q3 · Q5 — 匯出保留 / 設備問題
PART 2 · FAQ●
Q3 · 資料能匯出嗎?保留多久?
  • Trip / Event 清單:可匯出 CSV,日期範圍可選
  • 影片:保留 180 天,過期自動刪(每筆事件頁顯示倒數)
  • 想延長保留期 → 找你的服務窗口談
Q5 · 設備出問題我從哪裡看?
經銷商Master Portal → Diagnostics — 看跨車隊哪幾台有問題
車隊客戶Fleet Portal → Devices — 看自己車隊各台 ADAS / DMS 健康
Copyright © MiTAC Digital Technology Corp. 7 / 25

PART 3

Master Portal
7 頁速覽

經銷商 / 神達內部用 — 從設備派發到跨車隊稽核,你會用到的 7 個畫面。

Copyright © MiTAC Digital Technology Corp.
Master Portal · 1–4
PART 3 · 經銷商 / IT●
1. Dashboard
用途你管轄所有設備的即時總覽 能決定看到哪幾類問題在發生 → 分派人去處理 看到設備總數、已派發 vs 未派發比例、健康 vs 異常、5 種異常分類
2. Analysis
用途期間設備分配 + 訂閱方案演變圖 能決定看哪些方案賣得好 / 哪些車隊在升降級 看到過去一週(可調)的方案數量折線圖
3. Inventory
用途設備池(待分配 vs 已分配) 能決定把新進貨設備派給客戶,或把退回設備重派 看到設備清單 + IMEI / 機型 / 召回紀錄 / 連線時間
4. Diagnostics
用途跨車隊設備健康檢查 能決定鎖定異常設備,通知對應客戶或派 FAE 看到✓ 沒問題 / ⚠ 有問題 兩個數字 + 異常設備清單
Copyright © MiTAC Digital Technology Corp. 10 / 25
Master Portal · 5–7
PART 3 · 經銷商 / IT●
5. Fleets
用途你管轄的所有車隊清單 能決定新建車隊 / 加帳號 / 調整車隊資訊 看到車隊名 / 帳號數 / 設備數 / 創建日期
6. User Account
用途管理可登入 Master Portal 的人(經銷商員工) 能決定加新員工 / 停用離職員工 / 調權限(Admin / Viewer) 看到用戶名 / Email / 角色 / 啟用狀態
7. User Activity Logs
用途Master 層級登入登出稽核 能決定查可疑登入 / 跨組織存取 看到時間 / 用戶 / Email / 動作 / IP
Master Portal 的價值:你不用一個個進客戶 fleet 看 — 在這裡一眼看到「我管的所有設備」健康度、分配狀況、誰在用。
Copyright © MiTAC Digital Technology Corp. 11 / 25

Fleet · 每天看(2 頁)
PART 4 · 每天看●
Map Overview
用途即時車隊位置地圖 能決定看哪些車正在路上 / 偏離預定路線 看到世界地圖 + 設備 pin + 左側設備清單(可搜尋)
Dashboard
用途車隊一日總覽 能決定當日事件多不多?有沒有要立刻關注的駕駛? 看到Live Trips / Finished Trips / Unknown Drivers + 事件趨勢 + 駕駛安全評分排名
Copyright © MiTAC Digital Technology Corp. 14 / 25
Fleet · Trip 管理(3 頁)
PART 4 · TRIP●
Trip List
用途所有行程清單(預設過去 30 天) 能決定篩出特定車或駕駛的行程查證 看到每筆 trip 的時間 / 里程 / 駕駛 / 設備
Trip Detail
用途單筆行程的完整 timeline 能決定查這趟有沒有違規 / 異常事件 看到GPS 軌跡地圖 + 事件時間軸 + 調閱影片按鈕
Trip Report
用途期間行程量化報表 能決定衡量車隊整體活動量是否正常 看到Trip 數 / 總里程 / 總時長 + 趨勢圖 + 駕駛排名
典型工作流:客戶開 Trip List → 點某筆 Trip 進 Detail → 月底用 Trip Report 給主管看。
Copyright © MiTAC Digital Technology Corp. 15 / 25
Fleet · 事件 / 客訴(5 頁 / 1)
PART 4 · EVENTS●
Events 卡片牆
用途當日所有事件卡片牆 能決定快速掃過今天有什麼狀況 看到事件按類型分組(急煞 / 超速 / 跟車過近 等)+ 影片縮圖 + 影片保留剩餘天數
Event Detail
用途單一事件深入分析 能決定這事件要不要記過?要不要做駕駛訓練?要不要丟棄? 看到雙鏡頭影片 + 事件位置地圖 + 速度時間圖 + 加速度時間圖
客戶最高頻動作就在這兩頁。客訴調查、保險佐證、司機溝通都從這開始。
Copyright © MiTAC Digital Technology Corp. 16 / 25
Fleet · 事件 / 客訴(5 頁 / 2)
PART 4 · EVENTS●
File Retrieval
用途管理已調閱的影片檔案 能決定下載 / 整理事故證據 看到已調閱影片清單
Safety Reports
用途車隊安全月報 / 期間報表 能決定看趨勢有沒有改善,跟主管 / 保險彙報 看到駕駛 / 車輛 / 事件分類 排名與趨勢
Coaching
用途駕駛訓練教材庫 能決定挑選有教育意義的事件,跟司機坐下來檢討 看到被標記為「教材」的事件清單(從 Event Detail 標記過來)
Coaching 的小技巧:在 Event Detail 頁開「Add to coaching material」toggle,事件就會自動歸進這頁。司機培訓不用再翻檔。
Copyright © MiTAC Digital Technology Corp. 17 / 25
Fleet · 設備 / 駕駛 / 車輛(4 頁)
PART 4 · MANAGEMENT●
Devices
用途車隊內所有 K-series 設備清單 能決定看哪台設備需要 FAE 介入(ADAS / DMS Health) 看到設備 ID / 機型 / 上線狀態 / 韌體版本 / AI 健康度
Drivers
用途車隊駕駛員清單(綁 RFID 卡) 能決定加新司機 / 停用離職司機 / 看哪位司機開哪些車 看到司機名 / 員工編號 / IC 卡序號 / Email
Vehicles
用途車輛資料(綁設備) 能決定確認車牌 / 車型 / 設備配對 看到VIN / 車型 / 對應的設備 ID
Geofences
用途地理圍欄設定 能決定設定哪些區域進入 / 離開要警示 看到已建圍欄清單 + 地圖
Copyright © MiTAC Digital Technology Corp. 18 / 25
Fleet · 設定 + 行政(5 頁)
PART 4 · CONFIG / ADMIN●
Configurations
用途車隊全域設備行為調整 能決定依車型(轎車 / 中型 / 重型)調觸發門檻、音量、語音、超速安全分數 看到5 個 tab — Device Settings / Sensor Event / AI Event / Road Safety / Safety Score
Accounts & Permissions
用途Fleet Portal 帳號管理 能決定加管理員 / 設角色權限 看到帳號清單 + 角色 + 啟用狀態
Fleets
用途車隊本身的資料(logo / 時區 / 地址 / 電話)+ Contract Fleets 能決定更改車隊基本資料 看到Fleet 資訊表單
Device Usage
用途設備使用率 能決定看哪些設備閒置該回收 / 重派 看到每台設備使用紀錄
User Activity Logs
用途Fleet 內帳號操作稽核 能決定查可疑操作 / 違規授權 看到操作時間 / 用戶 / 動作
Copyright © MiTAC Digital Technology Corp. 19 / 25
